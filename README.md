
# react native sidemenu shrink example

Wanted to experiment with animations on android.


## Setup

```
$ npm install
```

## Running on emulator

Start the emulator and run:

```
$ npm start
```

## Running on Device

Connect device (with usb debugging):

```
$ npm run android
```
